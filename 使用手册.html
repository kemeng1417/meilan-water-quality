三、新建检测报告（核心操作）

步骤

  1. 点击导航栏「新建报告」
  2. 选择水样类型：出厂水 / 末梢水 / 直饮水 / 联合报告（出厂+末梢）
  3. 设置日期：
  4. 化验日期：默认为昨天（这是采样日期）
  5. 报告日期：默认为今天（这是出具报告日期）
  6. 提供「昨天」「今天」快捷按钮
  7. 录入化验员：系统会记住上次输入
  8. 选择采样点：勾选需要检测的点位
  9. 按区域分组折叠，可点击展开
  10. 每区域有「全选」「清空」按钮
  11. 支持搜索框过滤点位
  12. 支持「全选」「清空」「反选」
  13. 可保存/加载/删除选择模板
  14. 点击「创建报告」→ 弹出确认框 → 确认后生成报告

填报数据

操作 方式
填写检测值 直接点击单元格输入
Tab 跳格 Tab 跳到下一个单元格
Enter 换行 Enter 跳到下一行同列
整列填充 点击列头（如"肉眼可见物"）一键填充预设值
整列清空 点击列头右上角 × 图标
本行清空 点击行末「清空」按钮
全部清空 点击工具栏「清空数据」红色按钮
批量粘贴 点击「批量粘贴」，粘贴整张表格数据
撤销/重做 Ctrl+Z / Ctrl+Y

合规判定

  • 填入数据后即时显示合格（绿底）或超标（红底）
  • 结论列显示每行状态：○未填 / ◐部分 / ✓合格 / ✗超标

全屏填报

  • 点击工具栏全屏图标进入全屏模式
  • 按 Esc 退出全屏

四、保存与提交

保存数据

  • 点击「保存」按钮
  • 系统每 60 秒自动保存一次（底部状态栏有提示）
  • 底部状态栏显示：未保存（橙色）/ 已保存 14:30（绿色）

自动生成结论

  • 每次保存自动生成结论：
  • 全部填完且合格 → "本次检测项目全部合格"
  • 有未填项 → "已检项目均符合标准，尚有 N 项未填报"
  • 有超标 → "发现 N 项超标，涉及 N 个采样点……需整改"
  • 手动编辑结论后，系统不再覆盖

提交审核

  1. 确认数据填报完整、结论正确
  2. 点击「提交审核」按钮
  3. 提交后不可再修改

审核流程

化验员填报 → 保存 → 提交审核 → 审核人审核 → 通过
                                    ↘ 打回 → 化验员修改 → 重新提交

审核人操作： 1. 在报告页面填写「审核人姓名」 2. 点击「审核通过」或「打回修改」

被驳回的报告： - 化验员修改后点击「重新提交」 - 提交前可多次保存

五、记录列表

筛选与搜索

  • 按水样类型、状态、日期范围、异常标志筛选
  • 搜索：输入报告编号或化验员姓名

批量操作

  1. 勾选行前的复选框
  2. 点击「批量删除」（已审核记录自动跳过，管理员不受限）
  3. 点击「批量导出」导出为 Excel

状态流转

状态 颜色 可操作
草稿 灰 编辑、删除、提交审核
已提交 蓝 审核通过、打回
已审核 绿 仅查看（管理员可删除）
已打回 橙 编辑、保存、重新提交

六、趋势分析

  1. 选择水样类型
  2. 选择指标（可多选）
  3. 选择采样点（可多选）
  4. 选择日期范围（提供「今天」「近7天」「近30天」快捷按钮）
  5. 点击「查询」

图表功能： - 折线图/柱状图切换 - 拖拽缩放（dataZoom 滑块） - 异常标记（红色图钉标注超标点） - 下方展示数据明细表格

七、异常管理

系统自动记录所有超标项目，包括： - 超标报告编号 - 超标指标和检测值 - 标准限值 - 采样点信息

可录入整改变措施，标记异常是否已处理。

八、采样点管理

  • 查看所有采样点列表（按水样类型筛选）
  • 新增采样点：填写编码、名称、区域、楼层
  • 编辑/停用/启用采样点
  • 停用的采样点不出现在新建报告中

九、人员管理（仅主管）

  • 查看所有用户列表
  • 添加新用户：设置用户名、显示名、角色（主管/化验员）
  • 编辑用户信息
  • 停用/启用用户

十、报告导出

在报告编辑页面，点击「导出报告」下拉菜单，支持四种格式：

格式 说明 用途
Word (.docx) A4 横向，宋体小四，含照片 打印存档

十一、常见问题

Q: 为什么选完采样点显示 3/2？ A: 浏览器缓存了旧数据。按 F12 → 控制台 → 输入 localStorage.clear(); location.reload() 回车。

Q: 提交审核后还能改吗？ A: 不能。如需修改，请审核人「打回」后重新编辑提交。

Q: 已审核的报告能删除吗？ A: 普通化验员不能，需要主管（admin 账号）操作。

Q: 自动保存的数据在哪里？ A: 自动保存将数据写入数据库，60 秒自动触发一次。底部状态栏显示「已保存 HH:mm:ss」。

Q: 填错了怎么快速清空？ A: 列头 × 清空整列 / 行末「清空」清本行 / 工具栏「清空数据」全部清空，误操作可用 Ctrl+Z 撤销。

Q: 页面卡住无法输入？ A: 刷新页面（F5），未保存的数据会自动恢复。
